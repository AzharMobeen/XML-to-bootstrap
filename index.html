Welcome

There's no place like localhost

Welcome to the Demo pages for XML-to-Bootstrap. This site is generated from the sample files included in the source code.

To get started with your own content simply modify the source files in src/ directory and rebuild using grunt. Don't worry, your modifications will not be overwritten by future updates of XML-to-Bootstrap.

Demo pages use an unmodified version of Bootstrap — the default theme. Remember that all colors, fonts, etc. can be customized by simply modifying Bootstrap's scss/_variables.scss file. You will need to rebuild Bootstrap after doing so (in modules/bootstrap/), prior to rebuilding XML-to-Bootstrap.
