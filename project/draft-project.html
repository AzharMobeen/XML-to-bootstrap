This is a draft

Draft projects are not published. This means that they are not shown in the overview and are not linked in navigation elements. In addition, draft projects will be excluded from search engines by including a 'noindex' meta tag. However, pages are still being generated so that the content can be previewed.

A project is considered draft if there's an attribute 'draft' in the 'project' element.

Simply remove the 'draft' attribute once the project is to be published.
